# .vim
My vim configuration

## Getting Started

```sh
$ cd ~/
$ git clone --recursive https://github.com/cliffom/.vim.git .vim
$ ~/.vim/setup
```

## Vundle

[Vundle](https://github.com/VundleVim/Vundle.vim) is used to manage the
plugins.
